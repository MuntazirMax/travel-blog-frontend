# Top Destination Landing Page

A responsive travel website landing page built using semantic HTML5 and modern CSS.

## 🚀 Features

- Clean semantic structure (header, section, article, footer)
- Responsive layout using Flexbox and CSS Grid
- Optimized hero section
- Article card components
- SEO-ready meta structure
- Lightweight and fast loading

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Responsive Design Principles

## 🎯 Project Goal

This project focuses on building scalable UI structure, improving layout architecture skills, and understanding real-world landing page composition.

## 📌 Future Improvements

- Add JavaScript interactivity
- Implement search functionality
- Convert into a dynamic blog using Django
- Improve accessibility (ARIA, contrast ratios)
